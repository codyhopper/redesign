About this site

December 17, 2019

This site is a log of major design changes for my portfolio to track the evolution of my online presence. For now, my goal is to shift the focus of my portfolio site away from the “academic” and more into the “professional”.

I'm extremely excited to be creating this record. In personal projects, it’s so easy to blow through revision after revision. I'm looking forward to slowing down my process and being more reflective and open in my design practice.

The design of this website is kept to a minimum. With no JavaScript, plug-ins, or flexbox, it’s just a simple container for talking about what I’m working on. This site has some of the same visual language as my portfolio site, like the bold sans serif and rules, but it’s also extremely light using only using webfonts, websafe colors, and less than 200 lines of CSS. The typography is all here:


      h1 {
        font-family: Helvetica, Arial, sans-serif;
        line-height: 50%;
        font-size: calc(32px + .25vw);
        line-height: calc(0em + .2vw);
      }

      h2 {
        font-family: Helvetica, Arial, sans-serif;
        line-height: 50%;
        font-size: calc(22px + .25vw);
        line-height: calc(0em + .2vw);
      }

      p {
        font-family: Georgia, serif;
        font-size: calc(18px + .25vw);
        margin: calc(1em + .125vw) 0;
        color: #2c2c2c;
      }

That's it! Thanks for joining me on this new approach to my design process.
I'm looking forward to seeing where this goes.
